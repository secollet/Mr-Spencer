# Mr. Spencer

**Open Source Intelligence (OSINT) Investigation Tool**

A Streamlit-based web application for conducting open source intelligence investigations. No API keys required.

## Features

- **Username Hunter** — Check 200+ websites for username existence
- **Email Investigator** — Email verification, social detection, DNS lookups, related emails, pastebin search
- **Domain WHOIS** — Domain registration and DNS information
- **Google Dork Generator** — Generate targeted search queries
- **Social Discovery** — Find social media profiles from name/email
- **Reverse Image Search** — Generate reverse image search links for major engines
- **Wayback Machine** — Check archived versions of URLs
- **Video Finder** — Search for videos featuring a target individual across platforms

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment

Deployed on Streamlit Community Cloud.
